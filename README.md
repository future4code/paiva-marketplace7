### :heavy_check_mark: Features 
- [x] Cadastro de serviços
- [x] Adiciona serviços ao carrinho 
- [x] Remove produtos do carrinho
- [x] Ordenado por maior preço, menor ou por nome

  - [x] Preço mínimo e máximo
  - [x] Nome
